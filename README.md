# qrest-restaurant-management
## How to run?
1. Create an **application.yml** file in src/main/resources/ based on the provided **application-example.yml**.
2. Update _username_ and _password_ in the **application.yml** file to match your local database credentials.
3. Create a database named _qrest_management_ in your local database system.
4. Run the **QrestApplication** class to start the application.
